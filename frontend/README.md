# Mariech Autospare Frontend

Frontend e-commerce site for Mariech Autospare.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Product catalog with categories
- Product details pages
- About Us section
- Contact Us form
- Responsive design with Tailwind CSS
